# Myraid Secure Task Manager

## 🚀 Project Overview
This is a full-stack secure task manager application built as part of the Myraid internship assignment.

## 🔐 Features
- User Authentication (JWT + HTTP-only cookies)
- Password hashing using bcrypt
- End-to-end encryption (AES-256)
- CRUD operations for tasks
- Search, Filter, Pagination
- Secure APIs

## 🛠 Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB
